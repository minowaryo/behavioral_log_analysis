# Analysis 画面（週次・月次レポート） — 画面仕様メモ

## 対応UC
- UC-004: 週次の自動分析レポートを受け取る
- UC-005: 月次総合 AI 分析を受け取る
- UC-007: ビジネスアイデア提案レポートを受け取る

```

│  ┌────────────────────────┐  │

│  └────────────────────────┘  │
│                             │
```

## 注意点・設計意図

- レポートがまだ生成されていない場合は「次回レポート生成日: ○月○日（月曜日）」を表示する
- ビジネスアイデアの注記「これは可能性の提示であり、実在のビジネスを推奨するものではありません」は常時表示する（UC-007 業務ルール）
- 前回分析との差分表示は「新規」「継続」バッジで識別する（月次タブのアイデアのみ）
- 分析テキストは長文になるため、初期表示は折りたたみ（「もっと見る」で展開）
