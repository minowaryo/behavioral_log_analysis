# Settings 画面（YouTube 連携） — 画面仕様メモ

## 対応UC
- UC-002: YouTube 行動シグナルを同期する

## 画面構成

```
┌─────────────────────────────┐
│  [← 戻る]    Settings        │  ← ヘッダー
├─────────────────────────────┤
│  YouTube 連携                │
│  ┌─────────────────────────┐ │
│  │ 未連携                   │ │  ← 連携状態バッジ
│  │ [YouTube と連携する]      │ │  ← Google OAuth2 へ遷移
│  └─────────────────────────┘ │
├─────────────────────────────┤
│  ── 連携済み時の表示 ──       │
│  ┌─────────────────────────┐ │
│  │ ✓ 連携中                 │ │
│  │ 最終同期: 2026-04-18 07:00│ │
│  │ 取得シグナル数: 1,234件   │ │
│  │ [今すぐ同期]              │ │
│  │ [連携を解除する]          │ │  ← 解除してもシグナルは保持
│  └─────────────────────────┘ │
└─────────────────────────────┘
```

## 注意点・設計意図

- 「連携を解除する」タップ時は確認ダイアログを表示する
  - 「トークンのみ削除されます。これまでに取得したシグナルデータは保持されます」と明記する
- 同期中は「今すぐ同期」ボタンをスピナーに変えて二重実行を防ぐ
- 同期はバックグラウンドジョブで実行するため、完了通知はポーリングまたは画面リロードで確認する
